# sdd-lite CLI (local)

本仓库提供一个本地使用的 `sdd-lite` CLI，用于在项目中快速落地 SDD-lite（场景驱动开发-lite）的目录与骨架文件。

## 使用

- 直接运行：`node bin/sdd-lite.js --help`
- 初始化：`node bin/sdd-lite.js init`
- 自检：`node bin/sdd-lite.js doctor`
- 模块骨架：`node bin/sdd-lite.js add-module <modulePath>`
- 添加场景：`node bin/sdd-lite.js add-scenario <modulePath>`
